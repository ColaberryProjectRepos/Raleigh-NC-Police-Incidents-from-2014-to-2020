Extracting Raleigh Police Incidents Data 9


To download and extract the data required for this project, click on the icon on the top of this step.  The zip file will contain the following files & folders:

dataset folder - An Excel binary  file that has the data from 2014 to 2020 police incidents

DAX_Measures_Columns folder - files containing the measures and column formulas.

images folder - images that will be used

template - A Power BI template to get you started with the project.
